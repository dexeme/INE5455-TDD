
# Requisitos

# Como Executar


## Cobertura

Visualizar cobertura de comandos 

```
coverage run --branch -m unittest main.TestesTDD
coverage html
```


Visualizar cobertura de ramos 

```
coverage run --branch -m unittest main.TestesTDD
coverage html
```
